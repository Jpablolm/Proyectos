Proyecto "API_project"

Es un proyecto que tuvo como objetivo el extraer datos de una api en el cual se iban actualizando constantemente y luego ingresaros a una base de datos SQL para posteriormente realizar dashboards en Power BI con la información ingresada en la base de datos para visualizar estadísticas y gráficas de los datos extraídos que se iban actualizando constantemente.
